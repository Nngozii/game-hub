# GameHub

A modern web application for discovering video games, built with React. GameHub allows users to browse, search, and explore games across different genres, platforms, and ratings using real-time data.

## Features

* Search for games by title
* Browse games by genre and platform
* View ratings and game details
* Fast and responsive UI
* Fully responsive design

## 📦 Installation

1. Clone the repository:

```bash
https://github.com/Nngozii/game-hub.git
```

2. Navigate into the project directory:

```bash
cd gamehub
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
